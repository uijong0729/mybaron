# mybaron


개발 스펙
1.	서버 : Nginx (CentOS7)
2.	프레임워크 : Spring-boot 2.02
3.	DataBase : H2 DB → MYSQL   (6/12 : db안정성 문제로 MYSQL로 교체)
4.	ORM framework : JPA
5.	Template Engine : Mustache
6.	개발언어 : Java
7.	API : Riot API
8.	기타 라이브러리 : jQuery, bootstrap, Lombok, Riot-api-java 4.0.1(SDK), Spring Websocket
