개발 스펙
-------------
>	서버 : Nginx (CentOS7)
>	프레임워크 : Spring-boot 2.02

>	ORM framework : JPA
>	Template Engine : Mustache
>	개발언어 : Java
>	API : Riot API
>	기타 라이브러리 : jQuery, bootstrap, Lombok, Riot-api-java 4.0.1(SDK), Spring Websocket
